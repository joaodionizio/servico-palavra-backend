# Roadmap

## Feito nesta base

- Solution e projetos em camadas.
- Swagger/OpenAPI.
- Health check.
- CORS.
- Middleware global de excecoes.
- Padrao de resposta.
- EF Core com SQLite.
- Mappings Fluent API.
- Migration inicial.
- Seed de perfis e admin.
- Identity cookie auth.
- Provider PostgreSQL por configuracao.
- Rate limiting e hardening inicial de headers.
- Auth funcional.
- CRUD inicial de categorias, conteudos e trilhas.
- Favoritos e conclusao de conteudo.
- Estrutura inicial do plano biblico.

## Proximos passos

- Criar validacoes mais ricas por request.
- Expandir testes unitarios e integracao.
- Popular `BaseBiblica` com a ordem pastoral definitiva.
- Implementar geracao completa de dias/capitulos do plano biblico.
- Avaliar MFA e recuperacao de senha com tokens internos do Identity.
- Adicionar endpoints de administracao de usuarios.
- Preparar configuracao de disco persistente para Render.
- Decidir migracao para ASP.NET Core Identity.
